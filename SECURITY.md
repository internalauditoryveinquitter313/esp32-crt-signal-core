## Reporting a Vulnerability

This is an embedded firmware project that generates analog video signals. The attack surface is limited to physical
access and serial console.

If you find a security issue (e.g., buffer overflow in signal processing, unsafe DMA access, watchdog bypass), please
report it responsibly:

1. **Do NOT open a public issue.**
2. Email: **gabrielmaialva33@gmail.com**
3. Include: description, affected component, and reproduction steps.

You will receive a response within 72 hours.
